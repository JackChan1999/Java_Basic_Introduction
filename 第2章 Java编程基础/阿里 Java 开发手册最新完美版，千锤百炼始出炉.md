**升级亮点**

 

一、排版布局

​       1）页码中的封面、前言、目录并不计入总页码，调整前言到目录之前。

​       2）一级标题统一换页；二级标题统一不换页，并在之前空一行。

​       3）增加附1（版本历史）和附2（专有名词说明）并各自单独成为一页。

​       1）关于太阳年龄的示例错误。太阳年龄46亿，超过42亿的表示范围。
